Design a modern desktop application UI for a "MOA & Legal Opinion (LO) Tracking System".

Style:

* Clean, minimal, professional (similar to admin dashboard)
* Soft shadows, rounded corners
* Light theme with optional dark mode
* Use consistent spacing and alignment

---

MAIN LAYOUT

* Left sidebar (fixed width)
* Right main content area

Sidebar:

* Title at top: "MOA & LO Tracking System"
* Menu items:

  * Dashboard
  * Add Record
  * View Records
* Highlight active menu item
* Light/Dark mode toggle at bottom

---

DASHBOARD SCREEN

Top:

* Page title: "Dashboard Statistics"
* Subtitle: "Memorandum of Agreement and Legal Opinion Overview"

Content:

1. Statistic Cards (in a row):

* Total Records
* Ongoing Records
* Completed Records
* Missing Legal Opinion
* Missing Memorandum of Agreement

Each card:

* Icon
* Title
* Large number
* Rounded corners

2. Records Table (right side):

* Columns:

  * Control Number
  * School/University
  * Course
  * Status (badge style)
  * Workflow Stage

* Minimal design, clean rows

* Clickable rows

3. Small alert/info cards (left side):

* Missing Legal Opinion
* Missing MOA

---

ADD RECORD SCREEN

Layout:

* Two-column form inside a card

LEFT COLUMN:

* School/University (input with dropdown suggestion)
* Course (input with dropdown suggestion)
* Hours (input)
* Date Received (date picker)
* Status (dropdown)
* Workflow Stage (dropdown)

RIGHT COLUMN:

* Legal Opinion section:

  * Toggle: Upload File or Paste Link
* Memorandum of Agreement section:

  * Toggle: Upload File or Paste Link

Design:

* Each section inside rounded containers
* Clear labels and icons
* Clean spacing

---

VIEW RECORDS SCREEN

* Full-width table
* Search bar at top
* Filter dropdown (Ongoing, Completed, Missing LO, Missing MOA)

Table:

* Same columns as dashboard
* Clickable rows
* No action buttons

---

VIEW SINGLE RECORD SCREEN

Design like a clean form card (similar to mobile UI but for desktop):

Top:

* Control Number (e.g., MOA-2026-001)
* Back arrow (top right)

Fields (stacked vertically in rounded containers):

* School/University
* Course
* Hours
* Date Received
* Status
* Workflow
* Memorandum of Agreement
* Legal Opinion

Design details:

* Each field has an icon
* Inputs look like disabled form fields
* MOA and LO fields look clickable

---

INTERACTION DESIGN

* Clicking a record opens the detailed view
* Back button returns to previous screen
* Hover effects on clickable elements
* Smooth transitions

---

GOAL

Create a clean, modern, and professional UI suitable for a document tracking system used in an academic or government office.
